4. Беспроводные каналы передачи данных

Беспроводные каналы передачи данных используют для передачи сигналов электромагнитные волны, которые распространяются по эфиру.

Сотовая связь

  • Использует сеть базовых станций для связи мобильных устройств. Существует несколько поколений сотовой связи
    (2G, 3G, 4G, 5G), каждый из которых предлагает свои технологии и скорости передачи данных.(рис1)
    (рис1)

Wi-Fi

  • Беспроводные локальные сети (WLAN) для передачи данных на короткие расстояния.
    Wi-Fi основан на стандартных спецификациях IEEE 802.11 и широко используется в домах, офисах и общественных местах.(рис2)
    (рис2)

Bluetooth

  • Технология кратковременной беспроводной связи, предназначенная для передачи данных между устройствами
    на небольших расстояниях (обычно до 10-100 метров).(рис3)
    (рис3)

Спутниковая связь

  • Использует спутники для передачи сигналов на большие расстояния. Это особенно полезно в удалённых и труднодоступных районах.(рис4)
    • (рис4)

Радиосвязь

  • Используется для передачи данных на больших расстояниях,
    иногда через радиоволны. К ней относятся как аналоговые, так и цифровые системы.(рис5)
